# Evero Budget 💰

Evero Budget is a premium, high-security budget tracker designed for professionals who want smart insights and automated tracking.

## ✨ Features

### 🚀 Freemium Model
- **Starter Plan (Free)**:
  - Max 2 budget categories.
  - 12 transactions per month.
  - Manual data entry.
  - Standard login security.
- **Premium Plan ($9.99/mo)**:
  - **Unlimited** everything.
  - CSV & Bank imports.
  - **Mandatory 2FA Security** (Google Authenticator).
  - **Burn Rate Insights**: Advanced average spending trends.
  - Advanced charts and priority support.

### 🛡️ Security First
- Mandatory Two-Factor Authentication (2FA) for Premium users.
- Encrypted data handling.
- Login history tracking.

### 📊 Beautiful Visuals
- Dynamic glassmorphism UI.
- Real-time spending charts.
- Dark mode support.
- Interactive onboarding walkthrough.

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Shadcn UI
- **Database**: Prisma + SQLite
- **Auth**: NextAuth.js
- **Payments**: Stripe Checkout
- **Icons**: Lucide React

## 🚥 Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables (.env).
4. Push database schema: `npm run db:push`
5. Start developing: `npm run dev`
